To access the internet at SBRS 2014, select the WiFi SSID Stanford from your device's list of available wireless networks (note: this network is distinct from the "Stanford Visitor" network). Open a browser and load any URL; you will be redirected to an access page. There you may log in to the network through the "Sponsored Internet" option using the user name and password that will be provided at registration on the day of the Symposium.
